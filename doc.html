1. Installation

Just copy my code in suitable place.

  1. Open you adobe edge animate project
  2. Show the Code window (ctrl+E or cmd+E or Window->Code)
  3. Select Stage on the left hand column
  4. Copy following copy following code into it 
     var symbols;
 function getSymbolTree(sym){
     return [sym].concat(dfs(sym));
 }
 function control(symbolArray,funct){
     for (var i=0; i < symbolArray.length; i++) {
         eval("symbolArray[i]." + funct);
     }
 }
 function dfs(sym){
     var arr = sym.getChildSymbols();
     var ret = [];
     for (var i=0; i < arr.length; i++) {
         ret = ret.concat(dfs(arr[i]));
         console.log(arr[i].getSymbolTypeName() );
     }
     ret = ret.concat(arr);
     return ret;
 }
  5. Click the + button on the right hand side of stage, select Event and creationComplete
  6. Copy following code into it
     symbols = getSymbolTree(sym);

2. Documentation

Explain what and how my code work, if you are not interested in programming, just skip this part.

  1. Code in stage
    We put so-call global variable and function here, symbols is one of the example

    • symbols
      is a global variable stores every symbols on stages, include stage itself
    • getSymbolTree(sym)
      returns an array contains all child symbols of sym and sym itself.
    • control(symbolArray,funct)
      returns nothing.
      call the function funct for all symbols in symbolArray
    • dfs(sym)
      is a function for internal use, user should not call it.
      It performs dfs in sym and returns an array contain contains all child symbols of sym.
      Since sym is a tree, an array which used to mark the node is visited or not is not neccessary. This function is a recursive function, it is recommended NOT TO MODIFY THIS FUNCTION, unless you completely understand what you are doing.

  2. Code in creationComplete Every lines of code will be called, when everything is ready. So it is a good place to put getSymbolTree(sym) here

    • symbols = getSymbolTree(sym);
      fill symbols with an array contains all child symbols of sym and sym itself.

3. Demonstration

A few lines of code show that how to use my functions.

  1. Pause every symbols on the stages 
     control(symbols,"stop()");
  2. Play every symbols on the stages
     control(symbols,"play()");
  3. Reset all symbols on the stages
     control(symbols,"stop(0)");
  4. Play every symbols on the stages, start from 1 second
     control(symbols,"play(1)");
  5. To play an particular symbol named foo on the stage
    1. At stage, add following code 
       var foo;
    2. At creationComplete, add following code 
       foo = sym.getSymbol("foo");
 foo = getSymbolTree(foo);
    3. Add the following code in a suitable place 
       control(foo,"play()");
